# Симулятор цифровой связи (Streamlit)

Веб-версия симулятора цифровой связи с поддержкой модуляции M-PSK/M-QAM, канального кодирования (Хэмминг, Рида-Соломона), блочного перемежения и композитных каналов (AWGN, Релеевские/Райсовские замирания, импульсные помехи).

## 🚀 Запуск
```bash
pip install -r requirements.txt
streamlit run app.py
